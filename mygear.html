Bass

Pedals

Tuner

This tuner is really compact so it leaves space for more pedals and it is always on.

Drum pedals BeatBuddy and SDRUM

The beatbuddy is a must for bassists I think to have backing drums for creating solo work. The SDrum is great for when you have a riff and don't know what drum would suit without a drummer.

Octave Pedals Pog and /boss OC5

I had the OC5 first and when I first started using it I think it was helping my ear with the octave. Then I got the POG and used it for a bit, I'm back with OC5 currently. I interchange them.

Dirty Robot

I love the noises of this.

Bad Stone Phase Shifter

I like the noise of this one. Apparently Thin Lizzy used phasers so I like to think this helps my lizzy covers!

Bass Envelope Filter

I only got this pedal this year and its a favourite!

Compressor

I used to use this pedal a lot as I thought it was supposed to smooth the sound. I found it had a buzz and stopped using it lately.

JDI

I used to use a stagebug sb2 but I upgraded to this DI.

Pitch Shifter Brainwaves

This pedal I have set up for drop half step tuning.

Polara reverb pedal

Im not sure this was really a bass pedal but I love the sound and use it a lot.

Bass Soul Food

I have this pedal on frequently lately as the overdrive for the envelope filter.
